## Technical Notes

### Verification Methodology

1. Set location using `set_biblical_location()` or `set_custom_location()`
2. Set date using `set_time_gregorian()`
3. Focus on relevant object using `focus_on_object()`
4. Check eclipse parameters:
   - `umbral-eclipse-magnitude` > 0 = lunar eclipse in progress
   - `umbral-eclipse-magnitude` > 1 = TOTAL lunar eclipse
   - Sun-Moon angular separation < 0.5° = solar eclipse

### Pre-configured Events

All events are saved in `stellarium_server.py` PROPHETIC_EVENTS dictionary:
